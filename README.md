# Memoria Photobooth

This is a small Vite + React + Tailwind CSS demo recreating the attached design.

Quick start (PowerShell):

```powershell
cd f:\tania\web-tania\pkk
npm install
npm run dev
```

Notes:

- The project uses `tailwindcss` with `postcss` and Vite.
- Fonts are loaded from Google Fonts in `index.html`.
